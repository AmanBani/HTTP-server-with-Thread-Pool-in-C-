Features

  • Custom thread pool (mutex + condition_variable)
  • HTTP/1.1 request parsing
  • Static file serving
  • MIME type detection (HTML, CSS, JS, PNG …)
  • Path traversal protection
  • GET & HEAD support
  • Proper HTTP status codes (200 / 403 / 404 / 405)
  • SO_REUSEADDR for fast restart
  • Zero external dependencies

Build & Run

Compile (MinGW / GCC)
g++ -std=c++17 -O2 -Iinclude src/main.cpp src/ThreadPool.cpp src/HttpParser.cpp src/HttpHandler.cpp -o build/http_server.exe -lws2_32
Or with make
make run
Test with curl
curl -v http://localhost:8080/
