C++ HTTP Server

Built with a custom Thread Pool — no external libraries

Server Info

  • Language: C++17
  • Networking: Winsock2 (raw sockets)
  • Concurrency: Custom Thread Pool
  • Port: 8080

Architecture

Client Request
     │
     ▼
  accept()          ← main thread
     │
     ▼
 ThreadPool         ← enqueues handler
     │
     ▼
 parseRequest()     ← HttpParser
     │
     ▼
 buildResponse()    ← HttpHandler
     │
     ▼
  send()            ← back to client

Try it out

Open a terminal and run:

curl -v http://localhost:8080/
curl -v http://localhost:8080/style.css
